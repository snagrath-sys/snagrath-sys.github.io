WORK EXPERIENCE

ORACLE

In my stint at Oracle, even as a fresher I was given the sole
responsibility for the development and maintainance of a tool.
This tool generated millions of call sessions based on various protocols
like SIP,ISUP,ENUM,MEGACO, etc. Dealt with the nuances of each bit in each
layer of the OSI model. Coded mainly in C, some modules in Python and some json modules.
While working on the tool, made major contibutions by executing architectural changes
such as calls over TCP rather than UDP. Dealing with a code base as vast as 30 files,
Also enabled me to understand how huge projects are made and integrated and the working
of the indespensable wscript. Made the tool much more user friendly and
included individual requirements of each team member, since the tool was used by most colleagues.
Became adept at using tools like wireshark, GNU debugger, valgrind, etc. Also, understood
industrial style of coding and 3rd party library licensing. Pitched into Quality
Assurance, in order to help meet a stiff deadline, which also enabled me to get fair
understanding of the product end to end and from a utility perspective.
